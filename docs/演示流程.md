# LoyaltyLink 项目演示流程

## 环境准备

确保以下服务已启动：

| 服务 | 说明 |
|------|------|
| Ganache | 本地以太坊测试网，端口 `7545` |
| 前端服务器 | `http-server` 或 `serve`，端口 `3000` |
| MetaMask | 浏览器插件，连接 Ganache 网络 |

---

## 步骤一：启动服务

### 1.1 启动 Ganache

```bash
# 确保 Ganache GUI 或 CLI 正在运行，端口 7545
# 助记词：maid notable twist mutual dune speed come dolphin wet gaze scout sort
```

### 1.2 启动前端

```bash
# 在项目根目录执行
npx http-server frontend/ -p 3000
```

### 1.3 配置 MetaMask

1. 打开浏览器 MetaMask 插件
2. **添加网络**：
   - 网络名称：`Ganache Local`
   - RPC URL：`http://127.0.0.1:7545`
   - Chain ID：`1337`
   - 货币符号：`ETH`
3. **导入账户**：
   - 使用 Ganache 提供的私钥导入至少 **2 个账户**（商户 + 用户）

---

## 步骤二：演示流程

### 2.1 商户端 — 发放 ETH

1. 打开 `http://localhost:3000/merchant.html`
2. 点击 **"连接 MetaMask"**，选择商户账户（如 Account 1）
3. 确认页面显示商户地址和 ETH 余额（应有 100 ETH）
4. 在 **"发放 ETH"** 区域：
   - 输入用户的地址（如 Account 2 的地址）
   - 输入数量，如 `1`
   - 点击 **"发放 ETH"**
5. MetaMask 弹出确认窗口 → 确认交易
6. 页面显示 **"发放成功"**

**预期结果：** 商户余额减少，用户地址收到 1 ETH

---

### 2.2 用户端 — 查看余额和转账

1. 打开 `http://localhost:3000/index.html`
2. 点击 **"连接 MetaMask"**，选择用户账户（如 Account 2）
3. 确认页面显示用户 ETH 余额（应包含刚才收到的 1 ETH）
4. 在 **"转账 ETH"** 区域：
   - 输入任意接收地址
   - 输入数量
   - 点击 **"转账"**
5. MetaMask 弹出确认窗口 → 确认交易

**预期结果：** 转账成功，双方余额更新

---

### 2.3 商户端 — 扣款请求

1. 回到商户页面 `http://localhost:3000/merchant.html`
2. 在 **"扣款"** 区域：
   - 输入用户地址
   - 输入数量，如 `0.5`
   - 点击 **"生成扣款请求"**
3. 页面显示扣款请求详情：用户地址、商家地址、金额
4. **告知用户**：在用户端向商户地址发送对应金额的 ETH
5. 切换回用户页面，向商户地址转账指定金额

**预期结果：** 扣款请求信息正确展示，用户端可完成转账

---

## 演示要点

- **发放 ETH** 展示商户 → 用户正向资金流
- **转账 ETH** 展示用户间转账功能
- **扣款请求** 展示商户对用户的收款流程（类似积分消费/兑换）
- 所有交易均可在 MetaMask 中查看历史记录
- 可在 Ganache 中查看区块和交易详情
